## Suggested Title
**Desktop Data Automation and Financial Reconciliation Platform (Python/GUI)**

## Short Description (Summary)
I developed a complete desktop application in Python that automates complex administrative routines. The tool features an intuitive graphical interface and modules for financial reconciliation, intelligent PDF extraction, and Excel spreadsheet processing, reducing tasks from hours to a few seconds.

## Full Description

**The Problem:**
The client needed to handle a massive volume of data spread across Excel spreadsheets and PDF documents. Manual processing (such as financial reconciliation and order extraction) consumed hours of daily work and was prone to human errors.

**The Solution:**
I developed "AutoScript", a robust desktop application with a graphical user interface (GUI) built in Python. The tool centralizes multiple automation scripts into a single, easy-to-use executable.

**Key Features:**
*   **Automated Financial Reconciliation:** Cross-references thousands of Excel records with data extracted from PDFs, generating reports with conditional formatting to highlight discrepancies.
*   **Intelligent PDF Processing:** Reads order spreadsheets, searches for matches in PDFs with hundreds of pages, extracts the correct pages, and generates new consolidated documents.
*   **Advanced Excel Manipulation:** Automatically reorganizes and formats complex data using the Pandas library.
*   **User-Friendly Interface:** Allows users without technical knowledge to execute complex automations with just a few clicks.

**Technologies Used:**
*   **Language:** Python 3.8+
*   **Interface:** Tkinter (GUI)
*   **Data Processing:** Pandas, NumPy
*   **File Manipulation:** OpenPyXL (Excel), PyMuPDF/fitz (PDF)
*   **Distribution:** PyInstaller (standalone .exe generation)

**Results:**
The tool eliminated 100% of manual errors and reduced the processing time of daily routines from several hours to less than 1 minute, drastically increasing team productivity.
