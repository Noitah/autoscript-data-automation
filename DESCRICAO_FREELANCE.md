## Título Sugerido
**Plataforma Desktop de Automação de Dados e Conciliação Financeira (Python/GUI)**

## Descrição Curta (Resumo)
Desenvolvi uma aplicação desktop completa em Python que automatiza rotinas administrativas complexas. A ferramenta possui interface gráfica intuitiva e módulos para conciliação financeira, extração inteligente de PDFs e processamento de planilhas Excel, reduzindo tarefas de horas para poucos segundos.

## Descrição Completa

**O Problema:**
O cliente precisava lidar com um volume massivo de dados espalhados entre planilhas Excel e documentos PDF. O processamento manual (como conciliação financeira e extração de pedidos) consumia horas de trabalho diário e estava sujeito a erros humanos.

**A Solução:**
Desenvolvi o "AutoScript", uma aplicação desktop robusta com interface gráfica (GUI) construída em Python. A ferramenta centraliza múltiplos scripts de automação em um único executável fácil de usar.

**Principais Funcionalidades:**
*   **Conciliação Financeira Automatizada:** Cruza milhares de registros do Excel com dados extraídos de PDFs, gerando relatórios com formatação condicional para destacar divergências.
*   **Processamento Inteligente de PDFs:** Lê planilhas de pedidos, busca correspondências em PDFs de centenas de páginas, extrai as páginas corretas e gera novos documentos consolidados.
*   **Manipulação Avançada de Excel:** Reorganiza e formata dados complexos automaticamente usando a biblioteca Pandas.
*   **Interface Amigável:** Permite que usuários sem conhecimento técnico executem automações complexas com poucos cliques.

**Tecnologias Utilizadas:**
*   **Linguagem:** Python 3.8+
*   **Interface:** Tkinter (GUI)
*   **Processamento de Dados:** Pandas, NumPy
*   **Manipulação de Arquivos:** OpenPyXL (Excel), PyMuPDF/fitz (PDF)
*   **Distribuição:** PyInstaller (geração de arquivo .exe standalone)

**Resultados:**
A ferramenta eliminou 100% dos erros manuais e reduziu o tempo de processamento de rotinas diárias de várias horas para menos de 1 minuto, aumentando drasticamente a produtividade da equipe.
